# Postman API Testing

This folder will contain the exported Postman collection and environment
used to test the booking API.

The collection will include:

- Authentication requests
- Create-booking requests
- Retrieve-booking requests
- Update-booking requests
- Delete-booking requests
- Positive tests
- Negative tests
- Response validations

No live credentials, passwords, access tokens or private API keys will
be stored in this repository.
